## Usage

Clone the repository:

`git clone https://github.com/wvict/dominoes.git`

Enter the `dominoes` folder and compile the javascript files unsing the Typescript compiler:
```
cd dominoes
tsc
```

Go to `localhost:3000` and you are good to go!
